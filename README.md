IThreeM Developer Docs Guide Book


Run the following commands to start the local development environment:

## Usage

```bash
cd I3M-Developer-Docs
```
```bash
cargo install i3x-o1
```
- **Run Developer Docs Locally**:
  ```bash
  i3x-o1 run
  ```

- **Deploy to ICP**:
  ```bash
  i3x-o1 deploy
  ```
